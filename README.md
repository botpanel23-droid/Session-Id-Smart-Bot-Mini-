# 💎 CHALAH MD - WhatsApp Bot

## Deploy to Render (Free)

1. GitHub → New Repo → Upload all files
2. Render.com → New Web Service → Connect GitHub
3. Build: `npm install` | Start: `node index.js`
4. Add env vars: `PANEL_URL=https://your-app.onrender.com`
5. Deploy → Go to `/connect` → Scan QR
6. Login with number + OTP

## Features
- 🤖 AI Auto Reply
- 🎨 Image Commands (edit, sticker, blur, etc.)
- 📥 Social Media Download (YT, TikTok, FB, IG)
- 👥 Group Management (tagall, kick, warn, poll, etc.)
- 🚫 Anti Call, Anti Delete, One View Reveal
- 📊 Stats Dashboard
- 📣 Broadcast to all users
- ❤️ Channel React
- 🔧 Custom Commands (no code needed!)
- 💾 Auto Contact Save
- 🔄 Auto Update from GitHub
- 24/7 Self-ping for Render free tier
